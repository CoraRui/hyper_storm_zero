


This is a shmup, which alternates with a upgrade ui.

Upgrade Hangar

	parts shop - buy parts to use in the grid menu
	
	grid menu - place parts in the parts grid to upgrade ship
	
	upgrade hangar - buy general upgrades to the ship with money
	
	calendar - start the next day. there are three days total to get through
	
	
In game controls

directional keys to move

x to shoot

z to powerbomb(when you unlock them)

you have two additional lives. they persist across days
